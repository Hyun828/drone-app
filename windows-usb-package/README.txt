Drone App USB Package (Windows)
===============================

How to run
----------
1) Double-click run.bat
2) Browser opens automatically
3) If not opened, go to:
   http://127.0.0.1:8080/

Stop
----
- Close the command window, or press Ctrl+C

Required files in this folder
-----------------------------
- run.bat
- server.exe
- out\ (full folder, do not modify)
