제출 소프트웨어 안내
====================

연구보고서 제목 : 시뮬레이션으로 배우는 비행 — 드론 조종 코딩
대상 학년       : 초등학교 실과 5학년
(소프트웨어 시작 화면에 위 제목과 대상 학년이 표시됩니다.)

본 제출물은 동일한 소프트웨어를 두 가지 실행 형태로 제공합니다.

────────────────────────────────────────
[windows] 폴더 — 윈도우 PC 실행용
────────────────────────────────────────
  ├─ [media]    멀티미디어 교육자료 (비어 있음 — 사유는 폴더 내 README 참고)
  ├─ [program]  실행 파일
  │     run.bat 더블클릭 → 브라우저에서 자동 실행 (오프라인)
  │     index.exe = 컴파일된 로컬 웹서버, out\ = 웹앱(시작파일 index.html)
  └─ [source]   웹 애플리케이션(Next.js) 개발 소스 전체

────────────────────────────────────────
[app] 폴더 — 안드로이드 앱 실행용
────────────────────────────────────────
  ├─ [media]    멀티미디어 교육자료 (비어 있음 — 사유는 폴더 내 README 참고)
  └─ [source]   앱 실행파일 + 모든 소스
        ├─ 드론조종코딩.apk   (기기에 설치 후 아이콘 탭으로 실행, 완전 오프라인)
        ├─ android\           (안드로이드 앱 개발 소스)
        └─ web\               (앱이 내장하는 웹 애플리케이션 소스)

────────────────────────────────────────
참고
────────────────────────────────────────
- 화면 및 파일명에 시도명 / 학교명 / 출품자명은 포함되어 있지 않습니다.
- 본 소프트웨어는 미디어를 파일이 아닌 코드로 실시간 생성하므로 media 폴더는 비어 있습니다.
