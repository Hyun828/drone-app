[media] 폴더
============
이 폴더는 소프트웨어에 사용되는 멀티미디어 교육자료(사진/동영상/소리)를 저장하는 곳입니다.

■ 이 폴더가 비어 있는 이유
본 소프트웨어는 별도의 미디어 파일을 사용하지 않습니다.
모든 화면과 소리를 파일로 두지 않고 프로그램이 코드로 실시간 생성하기 때문입니다.

  - 드론, 장애물, 목표 지점, 바닥 등 모든 화면 요소
      → Three.js의 3D 도형으로 코드에서 실시간 생성 (이미지 파일 없음)
  - 버튼, 글자 등 화면 구성 요소
      → HTML과 CSS로 구성 (버튼 이미지 파일 없음)
  - 프로펠러 소리, 성공 효과음 등
      → Web Audio API로 실시간 합성 (음원 파일 없음)

따라서 image / movie / sound 에 해당하는 외부 파일이 존재하지 않아
이 폴더는 비어 있습니다.
