  초등학교 실과 5학년 · 시뮬레이션으로 배우는 비행
==========================================================

이 프로그램은 "웹 애플리케이션"으로 개발되었습니다.
(Next.js + React + Three.js 기반의 3D 드론 시뮬레이터)

그런데 왜 index.html 파일을 더블클릭해서 바로 실행하는
방식으로는 만들지 못했는지, 그 이유와 대신 마련한 실행
방법을 아래에 정리했습니다.


----------------------------------------------------------
1. 왜 index.html 더블클릭만으로는 실행되지 않는가?
----------------------------------------------------------

웹 앱을 정적 파일(out 폴더)로 내보내면 index.html이 생기지만,
이 파일을 브라우저에서 그냥 열면(file:// 방식) 정상 동작하지
않습니다. 이유는 다음과 같습니다.

(1) 절대경로 자원 로딩 문제
    - Next.js는 자바스크립트·CSS를 "/_next/..." 같은 절대경로로
      불러옵니다. file:// 로 열면 이 경로를 찾지 못합니다.

(2) ES 모듈 / CORS 보안 정책
    - 최신 웹앱은 자바스크립트를 "모듈(module)" 단위로 불러오는데,
      브라우저 보안 정책상 file:// 에서는 모듈 로딩과 자원 요청
      (fetch)이 차단됩니다. 그래서 화면이 비거나 3D가 안 뜹니다.

(3) 서비스 워커(오프라인 기능)는 http/https 에서만 동작
    - 오프라인 캐싱 같은 기능은 file:// 에서 등록 자체가 불가능합니다.

즉, "웹 앱"은 원래 "웹 서버(주소)"를 통해 열려야 정상 동작하도록
설계되어 있기 때문에, 단순히 index.html을 여는 것만으로는
실행할 수 없습니다.


----------------------------------------------------------
2. 그래서 어떻게 실행하도록 만들었는가? (오프라인 제출본)
----------------------------------------------------------

별도 프로그램 설치 없이, 클릭 한 번으로 실행되도록 두 가지
방식을 준비했습니다.

[윈도우 PC]  — windows/program/run.bat
    - 아주 작은 로컬 웹서버(index.exe)를 함께 넣었습니다.
    - run.bat 을 실행하면 내 컴퓨터 안에서만 도는 주소
      (http://127.0.0.1) 로 서버가 켜지고, 브라우저가 자동으로
      열려 프로그램이 실행됩니다.
    - 인터넷 연결이 없어도 동작합니다(완전 오프라인).

[안드로이드 / 태블릿]  — 모바일 APK 파일
    - 웹 앱을 앱(WebView) 안에 통째로 담은 APK로 만들었습니다.
    - 내부적으로 앱 전용 주소(https://appassets.androidplatform.net)
      로 자원을 제공하기 때문에, 위에서 말한 file:// 문제가
      발생하지 않습니다.
    - APK를 설치하면 홈 화면 아이콘을 한 번 탭하는 것만으로
      인터넷 없이 바로 실행됩니다(완전 오프라인).


----------------------------------------------------------
3. 학생들에게는 어떻게 실행시켰는가? (수업용)
----------------------------------------------------------

수업에서는 학생들이 설치 과정 없이 곧바로 사용할 수 있도록,
웹 주소로 접속하는 방식을 사용했습니다.

    - 개발한 소스 코드를 GitHub 저장소에 푸시(push)하고,
    - 이를 Vercel(웹 배포 서비스)로 자동 배포하여,
    - 아래 웹 주소로 누구나 바로 실행할 수 있게 했습니다.

      ▶  https://drone-app-virid.vercel.app/

학생들은 인터넷이 되는 PC·태블릿의 브라우저에서 위 주소만
입력하면 설치 없이 즉시 드론 시뮬레이터를 사용할 수 있습니다.


----------------------------------------------------------
4. 제출 파일 구성
----------------------------------------------------------

오프라인 환경(인터넷이 없는 교실·심사 환경)에서도 실행할 수
있도록, 제출 파일에는 다음 두 가지를 함께 넣었습니다.

    (1) 윈도우 실행 파일  — run.bat 으로 실행하는 PC용 오프라인본
    (2) 모바일 APK 파일   — 안드로이드 태블릿/폰용 오프라인 앱

정리하면,
    - 평소 수업    : 웹 주소(Vercel)로 간편하게 실행
    - 오프라인 제출 : 윈도우 실행파일 + 모바일 APK 로 실행
