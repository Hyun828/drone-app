USB 제출 패키지 안내
====================

연구보고서 제목 : 시뮬레이션으로 배우는 비행 — 드론 조종 코딩
대상 학년       : 초등학교 실과 5학년

본 USB 패키지는 오프라인 환경에서도 실행·심사할 수 있도록 구성했습니다.

────────────────────────────────────────
폴더 구성
────────────────────────────────────────

  [media]    멀티미디어 교육자료
      image\   사진·그림 (비어 있음 — README 참고)
      movie\   동영상   (비어 있음 — README 참고)
      sound\   소리·음원 (비어 있음 — README 참고)

  [program]  실행 파일
      windows\  윈도우 PC — run.bat 더블클릭으로 실행
      android\  안드로이드 — 드론조종코딩.apk 설치 후 실행

  [source]   웹 애플리케이션(Next.js) 개발 소스
      app\, public\, package.json 등 — 개발 환경 그대로
      (안드로이드 앱 소스는 program\android\ 의 APK로 제출)

  읽어주세요.txt — 실행 방식·기술적 배경 설명

────────────────────────────────────────
실행 요약
────────────────────────────────────────

  윈도우 PC  : program\windows\run.bat 더블클릭
  안드로이드 : program\android\드론조종코딩.apk 설치 후 아이콘 탭
  수업용 웹  : https://drone-app-virid.vercel.app/ (인터넷 필요)

────────────────────────────────────────
참고
────────────────────────────────────────
- 화면 및 파일명에 시도명 / 학교명 / 출품자명은 포함되어 있지 않습니다.
- 본 소프트웨어는 미디어를 파일이 아닌 코드로 실시간 생성하므로 media 폴더는 비어 있습니다.
