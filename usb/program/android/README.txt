[program/android] 폴더 — 안드로이드 앱 실행
==========================================

■ 실행 방법
  1) 드론조종코딩.apk 파일을 안드로이드 기기(태블릿·스마트폰)에 복사
  2) 기기에서 APK를 탭하여 설치 (출처를 알 수 없는 앱 허용 필요할 수 있음)
  3) 홈 화면 아이콘을 탭하여 실행 (인터넷 불필요, 완전 오프라인)

■ 구성 파일
  - 드론조종코딩.apk : 안드로이드 설치 파일

■ 참고
  웹 앱을 WebView 안에 담은 네이티브 앱입니다.
  앱 전용 주소(https://appassets.androidplatform.net)로 자원을 제공하므로
  file:// 문제 없이 오프라인에서 바로 실행됩니다.
