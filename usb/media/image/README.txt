[image] 폴더 — 사진·그림 파일
============================

이 폴더는 소프트웨어에 사용되는 사진·그림 교육자료를 저장하는 곳입니다.

■ 이 폴더가 비어 있는 이유
본 소프트웨어는 별도의 이미지 파일을 사용하지 않습니다.
모든 화면 요소를 프로그램이 코드로 실시간 생성하기 때문입니다.

  - 드론, 장애물, 목표 지점, 바닥 등 3D 화면
      → Three.js의 3D 도형으로 코드에서 실시간 생성 (이미지 파일 없음)
  - 버튼, 글자 등 화면 구성 요소
      → HTML과 CSS로 구성 (버튼 이미지 파일 없음)

따라서 사진·그림에 해당하는 외부 파일이 존재하지 않아
이 폴더는 비어 있습니다.
