[movie] 폴더 — 동영상 파일
============================

이 폴더는 소프트웨어에 사용되는 동영상 교육자료를 저장하는 곳입니다.

■ 이 폴더가 비어 있는 이유
본 소프트웨어는 별도의 동영상 파일을 사용하지 않습니다.
드론 비행과 코딩 학습 화면을 프로그램이 실시간으로 렌더링하기 때문입니다.

  - 3D 드론 시뮬레이션 화면
      → Three.js(WebGL)로 매 프레임 실시간 렌더링 (동영상 파일 없음)
  - 코딩 블록 조작·비행 결과
      → JavaScript로 즉시 계산·표시 (녹화 영상 없음)

따라서 동영상에 해당하는 외부 파일이 존재하지 않아
이 폴더는 비어 있습니다.
